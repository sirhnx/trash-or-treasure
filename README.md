# Trash or Treasure

Find valuable items in your collection instantly.

Upload a photo of CDs, records, books, or collectibles. The app uses AI to extract titles and searches Discogs for current market prices.

## Features

- 📸 Upload photos of collections
- 🤖 AI-powered title extraction
- 💰 Real-time pricing from Discogs
- 📊 Ranked by value
- 📱 Works on desktop and mobile

## How to Use

1. Visit the app
2. Upload a photo of your collection
3. Click "Analyze Collection"
4. Paste your Anthropic API key
5. See what's valuable!

## Getting Your API Key

Get a free Anthropic API key at https://console.anthropic.com/account/keys
